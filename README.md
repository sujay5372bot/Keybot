Advance Auto Message Forwarder Bot

Features:
- Public user login (OTP + 2FA)
- Auto message forwarding
- Keyword based rules
- Premium subscription system
- Weekly / Monthly / Lifetime plans
- UPI (manual), Razorpay & Telegram Stars
- Admin panel
- Auto premium expiry
- 24x7 VPS deploy ready

Setup:
1. pip install -r requirements.txt
2. Edit config.py
3. python bot.py
